## ğŸ› Troubleshooting

### Models not found error
```
ERROR: "models/linear_regression_reg.pkl not found"

Solution:
1. Run: python train_and_save_models.py
2. Wait for all models to train
3. Check that models/ directory exists with .pkl files
```

### Streamlit not installed
```
Solution:
pip install --upgrade streamlit
```

### Aquaculture.csv not found
```
Solution:
1. Ensure Aquaculture.csv is in same directory as app.py
2. File name is case-sensitive
```

### Port 8501 already in use
```
Solution:
streamlit run app.py --server.port=8502
```
